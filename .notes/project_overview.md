# FinHelper — обзор проекта

## Суть
Мобильный финансовый помощник. Пользователь ведёт доходы/расходы,
тегирует операции, загружает выписки из банков (PDF/QR),
ставит цели и следит за запланированными платежами.

## Ключевые экраны
- Dashboard — общая картина, баланс, последние операции
- Транзакции — список с фильтром по тегам
- Добавить операцию — форма + выбор тегов
- Цели — вклады и накопления
- Плановые платежи — подписки и регулярные траты
- Импорт — загрузка PDF или сканирование QR
